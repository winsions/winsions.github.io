iOS之谓词NSPredicate的应用

说到谓词，我们先来看一下谓词的语法。

  1. 比较运算符

    • >:大于
    • <:小于
    • >=:大于等于
    • <=:小于等于
    • =,==:等于
    • !=,<>:不等于

  2. 逻辑运算符

    • and /&&和
    • or/||或
    • not/!非
  3. 关系运算符
    • ANY任意，SOME 一些
    • ALL所有元素
    • NONE没有元素 等同于not any
    • in包含

  4. 范围运算符

    • between 如，1 BETWEEN { 0 , 33 }，或者$INPUT BETWEEN { $LOWER, $UPPER }。
    • in包含

  5. 字符串本身

    • SELF 如：@“self＝＝‘APPLEIOS’”
  6. 字符串相关
    • contain
    • between
    • endswith
  7. like通配符
    • like 如:@”name like[cd] ‘ios‘“ 
      @"name" like[cd] 'ios*'"
  8. 正则表达式matches
    • 如：NSString *regex = @”^A.+e$”; //以A开头，e结尾
      @”name MATCHES %@”,regex
  9. 数组操作
    • array[index]：指定数组中特定索引处的元素。
    • array[first]:制定第一个元素
    • array[last]:制定最后一个元素
    • array[size]:制定数组大小

4.正则表达式，例子是验证是否是手机号

//正则表达式串
NSString MOBILE = @”^1(3[0-9]|5[0-35-9]|8[025-9])\d{8}$”;
//创建含有正则表达式的帅选器
NSPredicate regextestmobile = [NSPredicate predicateWithFormat:@”SELF MATCHES %@”, MOBILE];
//筛选器的evaluateWithObject方法反向验证是否手机号，返回bool值
BOOL isPhoneNum=[regextestmobile evaluateWithObject:@”15300250500”] ;
